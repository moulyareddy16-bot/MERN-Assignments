# Week 6 - React & MERN Applications

This repository contains a collection of React and MERN applications developed during Week 6. These projects focus on advanced React concepts like Form Validation, Routing, Context API, and Full-Stack development.

## 📁 Projects Overview

### 1. 📝 User-Registration-App
A user registration interface that demonstrates robust form management.
- **Key Features:**
  - Form validation using `react-hook-form`.
  - Custom validation rules (whitespace checks, character limits).
  - Real-time data display in a structured table.
  - Automatic form reset upon successful submission.
- **Technologies:** React 19, Tailwind CSS 4, React Hook Form.

### 2. 🛣️ Tech-Routing-Portal
A demonstration of complex navigation patterns and layout management.
- **Key Features:**
  - Nested routing for sub-technologies (Java, Node.js, Vue).
  - Shared layout using `RootLayout` and `Outlet`.
  - Active link styling using `NavLink`.
  - Absolute path navigation to ensure stability across nested levels.
- **Technologies:** React 19, React Router v7, Tailwind CSS 4.

### 3. ⚛️ Context-State-Management
A demonstration of global state management without prop drilling.
- **Key Features:**
  - Shared counter state across 4 different components.
  - Context Provider pattern for centralized logic.
  - Interactive increment/decrement actions visible globally.
- **Technologies:** React 19, Context API, Tailwind CSS 4.

### 4. 🏢 EMP-APP: Full-Stack Employee Management
A comprehensive MERN stack application for managing employee records.
- **Key Features:**
  - Frontend built with React and Tailwind.
  - Backend API for CRUD operations.
  - Integrated database for persistent storage.
- **Technologies:** MongoDB, Express.js, React, Node.js.

## 🚀 How to Run

1. Navigate to any project directory:
   ```bash
   cd user-registration-app  # or tech-routing-portal, context-state-management, etc.
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```

## 🛠️ Global Technologies
- **Frontend Framework:** React 19 (Vite)
- **Styling:** Tailwind CSS 4
- **Routing:** React Router v7
- **Forms:** React Hook Form
