# Week 4: HTML Fundamentals and Structure

Welcome to the Week 4 assignment repository. This week focuses on mastering core HTML concepts, including semantic structure, nested lists, linking, and basic layout components.

## 🚀 Assignments Overview

This repository contains four key projects that demonstrate various HTML capabilities:

### 1. Introduction to Web Pages (`intro_to_web.html`)
- **Objective**: Explore the fundamental structure of an HTML document.
- **Key Features**: 
  - Usage of heading tags (`h1`) and paragraphs (`p`).
  - Implementation of basic text formatting (bold, italics).
  - Introduction to unordered lists for simple data representation.

### 2. Course List Catalog (`course_list.html`)
- **Objective**: Master the creation of complex, nested list structures.
- **Key Features**:
  - Implementation of nested unordered lists (`ul` within `ul`).
  - Organizing data hierarchically (e.g., Web Technologies > HTML, CSS, JS).
  - Clean semantic markup for improved readability.

### 3. Pochi the Cat: Personal Profile (`pochi_profile.html`)
- **Objective**: Build a multi-element personal profile page.
- **Key Features**:
  - Image integration with `alt` text for accessibility.
  - Combination of ordered (`ol`) and unordered (`ul`) lists.
  - Use of semantic horizontal rules (`hr`) for section separation.
  - External linking to high-quality media resources.

### 4. Black Goose Bistro Landing Page (`black_goose_bistro/`)
- **Objective**: Create a professional restaurant landing page with external styling.
- **Key Features**:
  - Directory-based project organization.
  - External CSS linking (`styles.css`) for presentation logic separation.
  - Use of `div` containers for logical section grouping.
  - Implementation of restaurant-specific information (Hours, Location, Catering).

## 🛠️ Technologies Used
- **HTML5**: Semantic markup and page structure.
- **CSS3**: Basic styling for the bistro landing page.

## 📈 Learning Outcomes
By completing these assignments, I have demonstrated proficiency in:
- Structuring content with semantic HTML elements.
- Organizing information using nested lists.
- Managing project assets using a clear directory structure.
- Integrating multimedia and external hyperlinks.
- Linking external stylesheets to HTML documents.
