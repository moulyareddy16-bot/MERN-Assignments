# Week 5 - MERN Stack Assignments

This repository contains the assignments completed during Week 5 of the MERN stack course. The focus this week was on advanced HTML tables, responsive layout design using Flexbox, and modern utility-first styling with Tailwind CSS.

## Assignments Overview

### 1. [Basic Table Structure](./Basic-Table-Structure/)
A foundational assignment demonstrating the use of `colspan` and `rowspan` to create complex, nested table layouts.

### 2. [Student Grade Report](./Student-Grade-Report/)
A practical application of HTML tables to display student performance across multiple subjects (Biology, Chemistry, Physics).

### 3. [User Contact List](./User-Contact-List/)
A styled directory showing user profiles. This project features a clean, responsive table design with circular profile images and hover effects.

### 4. [Beach Resort Cards](./Beach-Resort-Cards/)
A responsive card layout built using **CSS Flexbox**. It showcases a set of service cards with a cohesive design language and modern aesthetics.

### 5. [Tailwind Project Suite](./Tailwind-Project-Suite/)
A collection of professional UI components and pages built using **Tailwind CSS v4**.
- **User Testimonials**: A grid-based layout for customer feedback with gradient backgrounds.
- **Marketing Hero Page**: A high-impact jumbotron section with a clear call-to-action.
- **Organization Landing Page**: A branding-focused landing page with a hero section and feature cards.
- **City Travel Guide**: A comprehensive page layout featuring a navigation bar, hero image, and a multi-column footer.

## Technologies Used
- **HTML5**: For semantic structure and tables.
- **CSS3**: For custom styling, animations, and Flexbox layouts.
- **Tailwind CSS v4**: For fast, modern, and utility-first styling.
