# 🚀 MERN Stack Development - Week 1 Assignments

Welcome to my Week 1 assignments repository! This folder contains fundamental JavaScript logic exercises that lay the groundwork for building robust MERN stack applications.

## 📂 Project Structure

Each file in this directory addresses a specific logic-based problem using JavaScript.

| File Name | Description | Key Concepts |
|-----------|-------------|--------------|
| `greatestOfTwo.js` | Finds the maximum value between two numbers. | Ternary operators, Basic Logic |
| `greatestOfThree.js` | Identifies the largest of three given numbers. | Conditional Statements (If-Else) |
| `sumOfArray.js` | Calculates the total sum of all elements in a numeric array. | Array Iteration (for-of) |
| `minOfArray.js` | Determines the smallest number within an array. | Array Traversal, Comparison |
| `findMaxFunction.js` | A reusable function to find the biggest of three numbers. | Functional Programming, Arguments |
| `calculateSumFunction.js` | A function that accepts an array and returns its total sum. | Reusability, Array Processing |
| `linearSearchFunction.js` | Searches for a value in an array and returns its index or "NOT FOUND". | Linear Search Algorithm |
| `employeeManagement.js` | Advanced manipulation of an array of objects (Employee records). | Array Methods (splice), CRUD Logic |

## 🛠️ Technologies Used

- **JavaScript (ES6+)**: Core logic and data manipulation.
- **Node.js**: Environment used for execution.

## 🚀 How to Run

To execute any of these scripts, ensure you have Node.js installed and run:

```bash
node <filename>.js
```

Example:
```bash
node employeeManagement.js
```

## 📝 Learning Outcomes

- Mastery of JavaScript control structures (If-Else, Ternary).
- Proficiency in array manipulation and traversal.
- Understanding of functional programming and code reusability.
- Hands-on experience with object-oriented data structures in JavaScript.
